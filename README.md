# ProjektyKeZkousce
ProjektKeZkousce.zip je základní verze evidence pojištěných bez grafického rozhraní pro uživatele a možnosti ukládání údajů po vypnutí programu.
ProjektPlnaVerzeZaaklad již pracuje s HTML stránkami a databázi, ale neeviduje pojistné události a funguje jen pro roli admin.
První rozšíření přináší uživatelskou roli user a pojistné události. Entita admina nesmí být zároveň pojištěnec. Není to samozřejmě do detailu promyšlené jako ve veřejné apliakci, protože spoustu věcí může záležet na dohodě se zákazníkem.
Verze 2.5 přináší rozšíření rolí o pojistníky a pojištěné.
